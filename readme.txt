1. Set-up localhost by installing xampp(xampp is usally install in "C:\xampp").
2. Unzip demo.zip.
3. Save demo folder in "C:\xampp\htdocs".
3.1. rename the files in js folder change extension name from "txt" to "js"
4. To save database go to "http://localhost/phpmyadmin/".
5. Create new database by clicking Databases menu in menu bar.
6. Name the database "todolist".
7. Select the newly created database from the list of databases on the left side of the screen just below the text phpMyAdmin
8. Import the sql fild located in "C:\xampp\htdocs\Demo\db" from the saved folder of Demo.
9 If done correctly you can access the demo page using "http://localhost/demo/" as url.
